A simple helper to get started with selenium quickly

### Maven Dependency
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>selenium-helper</artifactId>
	<version>0.0.1</version>
</dependency>
```
